# Weather-app 🌦️

A simple and responsive Weather Application built using HTML, CSS, and JavaScript that fetches real-time weather information using the OpenWeatherMap API. Users can enter any city name and instantly view the current temperature, humidity, and weather conditions.

🚀 Features

🌍 Search weather by city name

🌡️ Displays current temperature in Celsius

🌧️ Shows humidity level

☁️ Displays live weather condition/description

⚡ Real-time data fetching using Fetch API

📱 Simple and user-friendly interface

❌ Error handling for invalid city names

🛠️ Technologies Used

 HTML5
 CSS3
 JavaScript (ES6)
 OpenWeatherMap API

Weather-App/

📌 How It Works

->User enters a city name.

->JavaScript captures the input.

->A request is sent to the OpenWeatherMap API.

->The API returns live weather data.

->The application displays:
->City & Country
->Temperature
->Humidity
->Weather Description
